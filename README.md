# simpod/skill-php

AI agent skill for idiomatic PHP development.

This repository contains one skill:

- `php`: general PHP typing, testing, style, and verification guidance.

## Install

With `skills`:

```sh
npx skills add simPod/skill-php --skill php
```

With `dotagents`:

```sh
npx @sentry/dotagents init
npx @sentry/dotagents add simPod/skill-php --name php
npx @sentry/dotagents install
```
